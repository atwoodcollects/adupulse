# Massachusetts Public Records Request Template
## For ADU Building Permit Data

### How to Use
1. Find the town's Records Access Officer (usually on town website under "Public Records" or "Town Clerk")
2. Email the request below
3. MA law requires response within 10 business days
4. Most requests are free; if they quote a fee over $20, you can negotiate or narrow the request

---

### Email Template

**Subject:** Public Records Request - ADU Building Permits

Dear Records Access Officer,

Pursuant to the Massachusetts Public Records Law (M.G.L. Chapter 66, Section 10), I am requesting the following records:

**All building permit applications for Accessory Dwelling Units (ADUs) submitted since February 2, 2025**, including:

1. Property address
2. Application/submission date
3. Approval, denial, or withdrawal date (and current status if pending)
4. Type of ADU (attached, detached, or internal)
5. Proposed square footage
6. Estimated construction value
7. Whether any variances or special permits were required

This request covers permits filed under the new statewide ADU law (Chapter 150 of the Acts of 2024, Section 8), which took effect February 2, 2025.

I am requesting this information in electronic format (Excel, CSV, or PDF) if available.

If any portion of this request is denied, please cite the specific exemption(s) and provide the non-exempt portions.

If the cost exceeds $20, please notify me before proceeding.

Thank you for your assistance.

Sincerely,
[Your Name]
[Your Email]
[Your Phone]

### What to Do If Denied or Delayed

1. **No response in 10 days:** Send follow-up citing the deadline
2. **Fee quoted:** Ask for itemized breakdown; negotiate scope
3. **Denied:** Request written explanation with specific exemption cited
4. **Appeal:** File with Supervisor of Records at Secretary of State's office

---

### Monthly Cadence

Send requests on the 1st of each month for the prior month's data. This builds a time series and keeps the data fresh.
